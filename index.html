Overview This is a SIMPLE bashdb debugger frontend. Useful for learning bash shell usage and writing simple scripts. Useful hint: shellcheck extension does a great job with finding common script errors before debugging. Usage Select Debug -> Add Configuration to add custom debug configuration (drop-down, path-input, etc...) Select Debug -> Start Debugging (F5) to start debugging See https://code.visualstudio.com/docs/editor/debugging for general usage. Sample features Debugging auto-configuration via launch.json
